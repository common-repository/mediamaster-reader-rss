RSS Reader for your site Animated !

== Description ==

This plugin provides the ability for administrators to include in wordpress platform, save and show in a post, 
the text widget or on a page, with a simple shortcode: [rssAnimated], the feed of any web service rss. 
Administrator can set a minimum number of recent posts that wants to display to users, 
set the profiles of the most popular social network, change colors and even choose three modes of animation, 
including flow velocity and pause.

Release 1.1 Update - Now you can set the colors of tags
Release 1.1.1 Update - Now you can to see all data and time for each post or podcast - Add tag audio player in podcast rss feeds
Release 1.1.2 Update - Now you can set the option to display the audio tag
Release 2.0 Update - Animated feeds and You can choose from 6 different options to extract your RSS feeds - visit this link to see plugin in action: http://ulmdesign.mediamaster.eu/jq_news_viewer/

== Installation ==

1. Upload folder `mediamaster-reader-rss` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the link in the admin section of your WordPress site, "AnimatedRSS" and set the options, choosing the url of the feed, the number you wish to display to your readers, style and more others options.
4. Copy and paste the shortcode [rssAnimated] in a post, page or widget-text where you want to show your feed.
